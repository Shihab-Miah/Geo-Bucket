Quick Setup Guide

Install in under 2 minutes. No developer account required.

  • Download & Extract
    Click download to get the ZIP archive, then Right-Click -> Extract All.
    প্রথমে জিপ (ZIP) ফাইলটি ডাউনলোড করুন এবং ফাইলটি Extract করে নিন।
  • Open Chrome Extensions
    Navigate to chrome://extensions in your browser URL bar.
    এবার ক্রোম ব্রাউজারের মেনু থেকে Manage Extensions পেজটি ওপেন করুন।
  • Enable Developer Mode
    Toggle the "Developer mode" switch in the top right corner.
    ডানপাশের উপরের কোণায় থাকা 'Developer mode' অপশনটি চালু করে দিন।
  • Load Unpacked
    Click the "Load unpacked" button and select your extracted folder.
    এরপর 'Load unpacked' বাটনে ক্লিক করে আপনার ফোল্ডারটি সিলেক্ট করুন।
