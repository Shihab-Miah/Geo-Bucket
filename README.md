## 🛠️ Installation Guide

Because Geo Bucket is a premium professional tool, it is installed directly from source to ensure you have the absolute latest version.

1. **Download the Extension:** Click the green **`Code`** button above and select **`Download ZIP`**.
2. **Extract:** Unzip the downloaded folder to a safe location on your computer.
3. **Open Chrome Extensions:** In Google Chrome, navigate to `chrome://extensions/`.
4. **Enable Developer Mode:** Toggle **`Developer mode`** ON in the top right corner.
5. **Load Unpacked:** Click **`Load unpacked`** and select the unzipped `Geo-Bucket` folder.
6. **Pin & Launch:** Pin the Geo Bucket icon to your toolbar, click it, and start sweeping the map!
